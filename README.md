# Shopping Floor Execution System
This is a Python project designed to manage and track inventory operations on a shopping floor. The repository contains modules to handle dispatching, tracking, and quality control processes.

## Project Structure

- `DISPATCH.py` - Handles dispatch operations.
- `Tracking.py` - Manages item tracking.
- `quality.py` - Performs quality control checks.
- `main.py` - Entry point for running the application.

## Getting Started

1. **Prerequisites**
   - Python 3.8 or higher installed on your system.

2. **Installation**
   ```bash
   # Clone the repository
   git clone <repository-url>
   cd Shopping_floor_Execution_System
   ```

3. **Running the Application**
   ```bash
   python main.py
   ```

## Usage

Each module can be executed or imported as needed based on your workflow. The `main.py` script ties together the core functionality.
